Formato

Los documentos HTML siempre tienen un mismo formato, estos empiezan por un inicio, este inicio contiene la declaración del documento y la definicion del idioma de este (Junto con el inicio del código).

Declaración del documento:

<!doctype html>

Definir el lenguaje por defecto:

<html lang=es>

Tras el inicio, debe continuar una cabecera (head), en la que se definen cosas como los metadatos, el enlace a un archivo de estilo, y el título de la página web en cuestión.

Esta cabezera se representará como:

<head> </head>

Después de la cabecera, se escribe el cuerpo del documento (body).

Este cuerpo se representará como:

<body> </body>

Por último, el código se finalizaría con el cierre de la etiqueta html:

</html>

Título de un Documento html

El título de un documento html se situará en el "head" de este, este título aparecerá como nombre de la pestaña al mostrarse en un navegador.

El el texto que será añadido como título será definido dentro de:

<title> </title>

Metadatos

Los metadatos dentro de un documento html tienen varios usos, como definir el formato de los carácteres del documento o para adaptar las palabras del documento según el dispositivo en el que se muestren.

Los metadatos no tienen que llevar una etiqueta que contenga una barra de cierre al terminar la línea (</meta> ).

Definir formato de los carácteres como UTF-8:

<meta charset="UTF-8">

Adaptar las palabras del documento segun el tamaño del dispositivo:

<meta name="viewport" content="width=device-width, initial-scale =1.0"

Definir el nombre del autor del documento. El nombre de autor que se quiera definir será escrito dentro de "content":

<meta name="author" content=" ">

Formato de Título

En html hay etiquetas especificas para los títulos, estas se se representan con una h y un número del 1 al 6, siendo 1 el formato predefinido de título más grande y 6 el más pequeño.

A la hora de escribirlo:

<h1></h1>

<h2></h2>

<h3></h3>

<h4></h4>

<h5></h5>

<h6></h6>

Secciones

Las secciones en un documento sirven para separar partes del código y aplicarles formatos específicos a los elementos que se situen dentro de cada seccion.

Para crear una sección:

<section> </section>

También se puede añadir un artículo, el cual tiene la misma función, pero esta centrado para englobar secciones del código más específicas.

Para crear un artículo:

<article> </article>

Agrupación

Párrafo

Para escribir texto en párrafos, estos se escriben dentro de la etiqueta "p".

Para crear un párrafo debajo de otro, se declara otro elemento con etiqueta "p" debajo del anterior.

Declaración de un párrafo:

<p> </p>

Texto Preformateado

El texto preformateado permite mostrar el texto escrito con el mismo formato que dentro de el documendo html.

Declaración de texto preformateado:

<pre> </pre>

Ejemplo de texto preformateado:

                <pre> Este
                        Texto
                            Aparecerá 
                                Tabulado</pre>

Listas

Se pueden crear tres tipos de listas de elementos dentro de un código html:

Listas Ordenadas

Las listas ordenadas,cada vez que se añaden elementos, estos se ordenan automaticamente por un número marcando su posición.

Ejemplo de una lista ordenada:

  1. Primer Objeto
  2. Segundo Objeto
  3. Tercer Objeto
  4. Cuarto Objeto

Formato de una lista ordenada:

<ol>
   <li>Primer Objeto</li>
   <li>Segundo Objeto</li>
   <li>Tercer Objeto</li>
   <li>Cuarto Objeto</li>
</ol>

Listas Desordenadas

Las listas desordenadas no generan ningun tipo de orden o número a los valores que tienen dentro.

Ejemplo de una lista desordenada:

  • Primer Objeto
  • Segundo Objeto
  • Tercer Objeto
  • Cuarto Objeto

Formato de una lista desordenada:

<ul>
   <li>Primer Objeto</li>
   <li>Segundo Objeto</li>
   <li>Tercer Objeto</li>
   <li>Cuarto Objeto</li>
</ul>

Listas de Definiciones

Las listas de definiciones almacenan valores junto con su correspondiente descripción.

Ejemplo de una lista de definiciones:

Primer Objeto
Este es el primer objeto
Segundo Objeto
Este es el segundo objeto
Tercer Objeto
Este es el tercer objeto
Cuarto Objeto
Este es el cuarto objeto

Formato de una lista de definiciones:

<dl>
   <dt>Primer Objeto</dt>
       <dd>Este es el primer objeto</dt>
   <dt>Segundo Objeto</dt>
       <dd>Este es el segundo objeto</dt>
   <dt>Tercer Objeto</dt>
       <dd>Este es el tercer objeto</dt>
   <dt>Cuarto Objeto</dt>
       <dd>Este es el cuarto objeto</dt>
</dl>

Línea Horizontal

La línea horizontal sirve para separar un párrafo del siguiente.

Formato de una línea horizontal:

<hr>

Elementos Semánticos de texto

Resaltado

El texto se puede resaltar de tres formas:

Negrita

Para que una parte del texto de un párrafo aparezca con letra en negrita.

<p> Aquí <b>hay</b> texto</p>

Como aparecería el texto:

Aquí hay texto

Subrayada

Para que una parte del texto de un párrafo aparezca con letra subrayada por debajo.

<p> Aquí <u>hay</u> texto</p>

Como aparecería el texto:

Aquí hay texto

Bastardilla

Para que una parte del texto de un párrafo aparezca con letra en cursiva.

<p> Aquí <i>hay</i> texto</p>

Como aparecería el texto:

Aquí hay texto

Salto de Línea

Para poder hacer un salto de línea en el párrafo de un archivo html se debe hacer un break dentro de la frase en la que se quiera saltar, en el mismo punto en el que se quiera hacer la separación.

Como insertar un salto de línea:

<p>Aquí <br>hay texto</p>

Como aparecería el texto:

Aquí
hay texto

Hiperenlace

Para poder insertar un hiperenlace, es necesario utilizar un contenedor tipo anchor(Ancla), el cual tendrá un atributo el cual indicará a que página web (Se utilizará la URL de la página web), directorio (Se utilizará el nombre del directorio seguido del nombre del archivo al que se quiere acceder) o elemento interno (Se referenciará al elemento utilizando un # al principio de su id) acceder.

El archor se creara de esta forma:

<a href=" "> </a>

Teniendo en cuenta que el contenido que se situará en a, ya sea un texto, una imagen,... Será el que se seleccionará a la hora de acceder al enlace proporcionado dentro de href.

Se pueden añadir dos atributos a el ancla.

Target, el cual hace que el archivo se abra en una pestaña nueva del navegador.

<a href=" " target=_blank> </a>

Download, el cual hace que en vez de acceder al elemento en cuestión, este se descarge.

<a href=" " download> </a>

Contenido Encajado, Imagenes

Se puede insertar contenido encajado como imagenes estáticas e imagenes animadas.

Para insertar una imagen se necesita especificar el archivo, dentro de un atributo src. Tambien se pueden añadir modificaciones en la altura (height) y anchura (width) de la imagen y una descripción a esta.

Insertar una imagen:

<img src=" ">

Insertar altura y anchura:

<img src=" " height= width= >

Insertar una descripción a la imagen:

<img src=" " alt=" ">

Tablas

El elemento de tabla representa información en más de una dimensión. Las tablas disponen de dos partes, igual que la estructura principal de un código html, la cabeza (thead) y el cuerpo (tbody).

Dentro de la cabeza, se colocan las tr (Table rows o lineas de tabla) y en estas, se colocan los títulos de cada columna de la tabla o th (Table header).

Dentro del cuerpo, tambien se colocan las tr, pero dentro de estas se situan los datos de la propia tabla.

Para poder unir dos columnas en una línea de la tabla, se utilizaría el colspan dentro de la linea en la que se quieran juntar, junto a la cantidad de columnas que se quieran unir. Para unir dos lineas, se utilizaría rowspan.

Formato de una tabla:

                <table border="1px">

                    <thead>

                        <tr>
                            <th>Producto</th>
                            <th>Número de Serie</th>
                            <th>Precio</th>
                        </tr>

                    </thead>

                    <tbody>

                        <tr>
                            <td>Ratón Logitech</td>
                            <td>332931</td>
                            <td>23.99€</td>
                        </tr>

                        <tr>
                            <td rowspan="e;2"e;>Teclado Razer</td>
                            <td>253687</td>
                            <td>74.50€</td>
                        </tr>

                        <tr>
                            <td>266402</td>
                            <td>42.05€</td>
                        </tr>

                        <tr>
                            <td>Fuente de alimentación Corsair</td>
                            <td>313039</td>
                            <td>80.00€</td>
                        </tr>

                        <tr>
                            <td>Cargador Nokia</td>
                            <td>274494</td>
                            <td>11.95€</td>
                        </tr>

                    </tbody>

                </table>

Esta tabla se verá en el documento como:
